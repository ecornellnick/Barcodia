# AI-Assisted Production Workflow

## Important
This prototype contains rough storyboard pages, not final illustrated comic pages.

## Recommended workflow per page

1. Start with the page SVG rough.
2. Generate or draw one clean page based on the page notes.
3. Use the locked references for character/environment consistency.
4. Save the final art as:
   - `chapters/chapter-01-youre-late-again/final_pages/page-001.png`
5. Update `js/chapter-data.js` image path from:
   - `pages/page-001.svg`
   to:
   - `final_pages/page-001.png`
6. Reopen `index.html` or refresh the browser.

## First anchor visuals to create

Create these before attempting every panel:

1. Hero in bed at 10:22 using the bedroom reference.
2. Phone close-up with sister texts.
3. Mother in kitchen saying “Late again?”
4. Hero running through the village.
5. Child scam moment.
6. Street punk blockade.
7. First skirmish resolution.

## Prompt discipline

Every final-art prompt should include:

- Which canon character reference is being used
- Which environment reference is being used
- Page number and panel number
- Shot type
- Facial expression
- Exact action
- Dialogue placement if needed
- Smooth clean anime/JRPG comic style
- Avoid rough/grainy texture

## Do not do this yet

Do not batch-generate all 10 final pages at once. Make Pages 1–3 first, test readability, then continue.
