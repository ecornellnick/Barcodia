# Start Here — Barcodia Comic Reader Prototype v0.3

Open `index.html` in your browser.

This is a local browser comic reader for the first Barcodia comic opening sequence, “You’re Late Again.”

## What is included

- A working local web comic reader
- Active PNG comic pages for Pages 1–10
- Page notes, panel notes, dialogue beats, and references
- Canon reference folders
- Markdown storyboard docs
- Starter layout templates
- AI-assisted production workflow notes
- Page replacement instructions in `docs/HOW_TO_ADD_OR_REPLACE_PAGES.md`

## Important path rule

The reader points to PNG files in:

```text
chapters/chapter-01-youre-late-again/pages/
```

So replacing `page-006.png` with a new revised `page-006.png` will update Page 6 in the reader.

## Current story status

Pages 1–10 cover the full current opening mini-arc, from the symbolic nightmare through the first street skirmish.

## Logo note

Some generated pages still include the old placeholder Barcodia icon. That symbol is deprecated and should not be used in future generation.
