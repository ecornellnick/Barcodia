# v0.6 Reader Update

This update adds:
- full-screen page browser modal
- thumbnails grouped by chapter
- go-to-page from main reader and from the modal
- remembered last-read page state using localStorage
- `P` keyboard shortcut to open/close the page browser
- `Esc` to close the modal

## Current content
- Chapter 1: pages 1–17

## Future chapter support
To add another chapter later, extend `js/chapter-data.js` inside `BARCODIA_BOOK.chapters`.
