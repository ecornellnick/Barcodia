# Barcodia Comic Layout Templates — Starter Set

Use these as starter archetypes. Build more templates only when a scene actually needs them.

## 1. Symbolic Nightmare Page
- 3–4 panels
- Broken or asymmetrical layout
- Minimal dialogue
- Visual motifs only
- Best used for dreams, foreshadowing, memory fragments

## 2. Warm Wake-Up / Establishing Page
- 4–5 panels
- One large environment panel
- Several close-ups for mood
- Best used to move from atmosphere into character

## 3. Text / Phone Realization Page
- 5–6 panels
- Tight close-ups and reaction panels
- Text bubbles carry relationship tone
- Best used when messages replace spoken dialogue

## 4. Scramble / Momentum Page
- 5 panels
- Slightly angled action panels
- Short dialogue only
- Best used to accelerate pacing without losing readability

## 5. Family Warmth Beat
- 3–4 panels
- Establish location, show reaction, exit on motion
- Best used for tiny lived-in emotional beats

## 6. Town Movement Page
- 4 panels
- Wide establishing panel + movement details
- Best used to show world while moving plot forward

## 7. Con / Interruption Page
- 5 panels
- Setup, interruption, appeal, hero reaction, hero choice
- Best used to reveal character through action

## 8. First Combat Page
- 5–6 panels
- Clear action/reaction/counter/resolution
- Avoid too many tiny panels until final art workflow is stable
