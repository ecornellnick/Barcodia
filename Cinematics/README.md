# Cinematics Folder

Place runtime-test cinematic MP4 files here.

Current Dev Mode viewer expects:

```text
Opening Cinematic Latest.mp4
```

The backend serves this folder at:

```text
/cinematics/<filename>
```

Example:

```text
/cinematics/Opening%20Cinematic%20Latest.mp4
```

Large MP4s are not included in ChatGPT patch zips unless explicitly uploaded for packaging.
